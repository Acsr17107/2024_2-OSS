# 2024년도 2학기 오픈소스 소프트웨어
## OSS 교과목 내용 (Git & GitHub) 정리

---

### Git 명령어 기초
- **`git --version`** : Git 버전 확인
- **`git config --list`** : Git 설정 확인

---

### 깃 설정 명령구조
- **`git config --설정범위 설정변수 설정값`**

#### 설정범위
- **`--system`** : 모든 사용자
- **`--global`** : 모든 저장소에 적용 (전역 설정)
- **`--local`** : 현재 저장소에만 적용 (로컬 설정)
